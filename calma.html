Calma

Diseño y desarrollo de una aplicación de meditación mindfulness

Proyecto conceptual desarrollado desde Figma hasta su implementación en HTML, CSS y JavaScript, enfocado en crear una experiencia digital simple, accesible y relajante.

Contexto

Calma es un proyecto personal cuyo objetivo fue diseñar y desarrollar una interfaz centrada en la meditación mindfulness.

El propósito principal fue poner en práctica habilidades de diseño UI/UX y desarrollo frontend, construyendo una experiencia coherente desde la idea inicial hasta el producto final.

Objetivo

El objetivo del proyecto fue crear una interfaz que permitiera a los usuarios comenzar una sesión de meditación de forma rápida, eliminando fricciones y manteniendo una experiencia visual calmada.

Proceso de diseño

El proceso comenzó en Figma, donde se definió la estructura visual, la jerarquía de contenidos y el sistema de diseño.

  • Definición de wireframes
  • Diseño de interfaz visual
  • Selección de paleta de colores y tipografía
  • Iteración sobre componentes UI

Decisiones de diseño

  • Uso de colores suaves para transmitir calma
  • Tipografía limpia para mejorar la legibilidad
  • Espaciado amplio para evitar sobrecarga visual
  • Interfaz minimalista centrada en el contenido

Desarrollo

Una vez definido el diseño en Figma, se llevó a cabo la implementación utilizando HTML, CSS y JavaScript.

  • Maquetación semántica con HTML
  • Estilos organizados y reutilizables con CSS
  • Interacciones y dinamismo con JavaScript

Responsive design

La interfaz fue adaptada para diferentes dispositivos, asegurando una experiencia consistente en escritorio, tablet y móvil.

Interacciones

Se incorporaron pequeñas interacciones y animaciones para mejorar la experiencia del usuario sin romper la sensación de calma.

  • Transiciones suaves
  • Animaciones sutiles
  • Feedback visual en elementos interactivos

Resultado

El resultado es una interfaz minimalista, funcional y visualmente coherente que refleja una experiencia centrada en el bienestar del usuario.

Este proyecto me permitió reforzar habilidades tanto en diseño como en desarrollo frontend, especialmente en la traducción de un diseño desde Figma a código real.

Conclusión

Calma representa un ejercicio completo de diseño y desarrollo, enfocado en la creación de interfaces limpias, accesibles y orientadas a la experiencia del usuario.
